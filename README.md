# Payroll Fraud Detection Dashboard

This project analyzes HR and payroll data to detect potential financial manipulation and suspicious payroll activities.

## Problem
The company suspected abnormal increases in payroll expenses and possible financial irregularities.  
The objective was to investigate employee records and identify anomalies.

## Dataset
The dataset contains information about employees including:

- Employee Name
- Department
- Employment Status
- Years of Experience
- Base Salary
- Bonus
- Overtime
- Total Expense
- Performance Scores

## Key Findings

- 1000 employee records analyzed
- 608 duplicate employee names detected
- 86 inactive employees still receiving payments
- $31M total payroll expense
- $3M flagged as suspicious expenses

## Analysis Performed

- Duplicate employee detection
- Status vs expense analysis
- Department expense analysis
- Bonus and overtime pattern analysis
- Salary vs Total Expense outlier detection

## Tools Used

- Power BI
- Data Analysis
- Financial Analysis
- Data Visualization

## Outcome

A Power BI dashboard was created to help management:

- Detect ghost employees
- Identify suspicious expenses
- Monitor department payroll distribution
